# Вирус для платформы Android
### Описание проекта

Моя цель — разобраться, как работают современное зловредное ПО. И лучший способ это сделать — попробовать создается похожий софт самому. При желании мой троян сможет наблюдать и передавать информацию о целевом устройстве на сервер.

Предполагаемые возможности трояна будут следующими:
- сбор информации о местоположении;
- получение списка установленных приложений;
- получение СМС;
- запись аудио;
- съемка задней или фронтальной камерой.
